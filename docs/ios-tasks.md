# Задачи: iOS приложение MusicPlay (релиз только для iPhone 16 Pro)

## P0 — Критические
- Реализовать токен‑based auth через `/api/v1`.
- Реализовать потоковое воспроизведение через `/api/v1/stream`.
- Реализовать поиск и воспроизведение результатов.
- Реализовать синхронизацию плеера с `/api/v1/player/state`.

## P1 — Важные
- Реализовать плейлисты (CRUD + reorder).
- Реализовать избранное (optimistic UI).
- Реализовать очередь (add/remove/reorder).
- Реализовать мини‑плеер и полноэкранный плеер.

## P2 — Улучшения
- Полировка UI в стиле Apple Music (HIG).
- Оптимизация списков (LazyVStack, кеш изображений).
- Улучшение стабильности фонового воспроизведения.
